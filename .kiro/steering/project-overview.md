# TNR Tracker Project Overview

## Project Purpose
This is a TNR (Trap-Neuter-Return) tracking application for managing community cat populations and TNR activities.

## Technology Stack
- **Frontend**: React with Vite
- **Backend**: Supabase
- **Deployment**: Vercel
- **UI Components**: Custom component library

## Important Guidelines
⚠️ **DO NOT modify existing code structure - the application is working and stable**

## Project Structure
- `src/` - Source code for the React application
  - `pages/` - Page components like Dashboard
  - `components/` - Reusable UI components
- `docs/` - Project documentation
  - API.md - API documentation
  - ARCHITECTURE.md - Architecture overview
  - DATABASE.md - Database schema
- `public/` - Static assets
- `.env` - Environment variables (contains secrets)

## When Working on This Project
1. Always read existing code before making changes
2. Follow the established patterns in the codebase
3. Test changes carefully to avoid breaking working functionality
4. Refer to documentation in the `docs/` folder for context
5. Respect the existing file organization
